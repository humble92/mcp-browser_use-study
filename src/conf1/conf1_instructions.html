Manual Login for Cookie Generation

This browser window has been opened to help you generate authentication cookies for later use.

Your Action Required: Please navigate to the website you need to log into (e.g., by typing the URL in the address bar above) and complete the login process manually within this window.

Once you have successfully logged in, please return to the terminal window from which you ran this script and press the Enter key.

After you press Enter in the terminal, your login session information (cookies) will be automatically saved to a file named cookies.json in the root directory of this project.

This cookies.json file can then be used by other operations to access the website as if you were logged in, without needing to log in again each time.

IMPORTANT: Do NOT close this browser window manually. After logging in, return to your terminal and press Enter. The script will then save the necessary cookie information and close this browser window automatically.

Thank you for your cooperation!
